## 1. GitHub Desktop

1. [GitHub Desktop](https://desktop.github.com/)のサイトをアクセスしてダウンロードする。
2. インストールする。
3. 開いた画面で、「`Sign in to GitHub.com`」をクリックする。
4. ユーザー名とパスワードを入力して、「`Sign in`」ボタンをクリック。
5. 次の画面では何も変更せずに、「`Continue`」ボタンをクリック。
6. チェックマークを外して、「`Finish`」
